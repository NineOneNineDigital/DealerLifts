# Turn14 Fitment Filter Allowlist — Trucks, SUVs & UTVs/Side-by-Sides

Paste target: **Data Here-to-There Turn14 app → Filters tab → fitment filter** (Make → Model → SubModel → Year → **Add**).

Scope: US-market **trucks + SUVs + UTV/side-by-side (powersports)**. Excludes passenger cars, import/euro/street-performance tuner platforms. Universal-fitment parts (winches, recovery, lights) are **not** governed by this filter — keep "Exclude universal fitment parts" **OFF** and clean those up downstream.

How to use:
- Leave **SubModel** blank and **Year** as the full range unless you want to cap a platform (e.g., classics).
- App dropdowns are driven by Turn14's VCdb vehicle DB — if a name below isn't an exact match, pick the closest option shown.
- **Start with Tier 1** (highest aftermarket volume). Expand into the full list as time allows.
- Crossovers / soft-roaders and ATVs are marked _(optional)_ — include only if you want to cast wider.

- **Universal offroad gear** still needs a downstream filter (Categories/Brands or a storefront backstop) — the fitment allowlist can't touch no-fitment parts.
